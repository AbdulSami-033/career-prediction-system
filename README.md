# Career Prediction System

A machine learning-based web application that predicts suitable career paths based on user skill levels.

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- KNN Algorithm

## Features
- Interactive skill assessment
- Career prediction using KNN
- Top 3 career recommendations
- Confidence score visualization

## How to Run
1. Install dependencies
2. Run:
   streamlit run app.py
